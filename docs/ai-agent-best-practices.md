# Bonnes pratiques pour construire des agents IA

Construire un agent fiable nécessite plusieurs bonnes pratiques.

## Limiter le périmètre

Un agent doit avoir un objectif clair.

## Contrôler les actions

Les appels d’outils doivent être contrôlés.

## Journaliser les actions

Les systèmes d’agents doivent être observables.

## Ajouter des validations humaines

Certaines actions doivent être validées par un humain.
