# Glossaire des agents IA

## Agent IA

Un agent IA est un système basé sur un modèle de langage capable d’exécuter des actions pour atteindre un objectif.

## Orchestration

L’orchestration correspond à la coordination des différentes étapes d’un système d’agents.

## Tool calling

Mécanisme permettant à un agent d’utiliser des outils externes.

Exemples :

- API
- base de données
- moteur de recherche

## Mémoire

La mémoire permet à un agent de conserver des informations entre plusieurs étapes.

## Workflow

Un workflow est une séquence structurée d’actions automatisées.
